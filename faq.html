40K Scenario and Objective API

Welcome at the 40K Scenario and Objective API

You can use this api fto find scenarios and objectives for your 40K games, or use it for apps.

The following methods are allowed:

  • GET

The following resources are available:

  • scenarios
  • objectives

Using the following URI's:

  • /scenarios
  • /scenarios/id, "/scenarios/1"
  • /objectives
  • /objectives/id, "/objectives/11"

Objectives are numbered on a D66 basis.
